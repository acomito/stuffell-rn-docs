UserCard

A card to display a user Avatar, full name, and Follow/Unfollow button. Used in contructing the results for a UsersSearch component.
Properties:
Name Type Description
targetUser object the target user is the person displayed on the user card.
currentUser object The currently logged in user. we use properties from this object to match against the target user and answer questions like " are these peope already friends?"
